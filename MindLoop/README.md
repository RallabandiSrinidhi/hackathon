# MindLoop Prototype

This is a lightweight UI prototype built with React and Vite. It demonstrates basic screens and navigation for a student learning app.

## Features

* Login screen (no real auth)
* Home screen with navigation buttons
* Upload notes screen (shows file name)
* Summary screen with dummy text
* Quiz screen with hardcoded questions
* Flashcards screen with flip cards

## Running

1. `npm install`
2. `npm run dev`

Open the URL shown by Vite (usually http://localhost:5173)

This is a hackathon demo; functionality is minimal and uses placeholder data.
